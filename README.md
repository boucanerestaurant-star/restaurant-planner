# Restaurant Planner

## Introduction
Welcome to the Restaurant Planner project! This application helps you design and plan restaurant layouts efficiently. 

## Features
- **60-seat capacity**: Plan and visualize seating arrangements for 60 guests.
- **Dining Room Size**: A spacious 60m² dining room.
- **Kitchen/Fireplace Area**: A dedicated 40m² kitchen area for cooking and a fireplace for ambiance.
- **LocalStorage Persistence**: User settings and layouts are saved in LocalStorage for easy retrieval.
- **Interactive Editor**: Built with [Konva.js](https://konvajs.org/), the interactive editor allows you to drag and drop elements for easy layout design.
- **Template Selector**: Choose from 5 predefined layouts to kickstart your design.

## Installation Instructions
To install the project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/boucanerestaurant-star/restaurant-planner.git
   cd restaurant-planner
   ```
2. Install the dependencies using npm:
   ```bash
   npm install
   ```
